github 사용법
1. web 상의 git과 연결될 local folder 생성 (한번만)
2. 연결 (한번만)
git init
git remote add origin repository address
3. git pull origin master : 땡겨오기
4. push
git status : 상태 확인
git add . : 모두 올리기
git commit -m "massage" : 간단한 설명문
git push origin master : 보내기
